 ◦ 핵심 기능
   -상담 신청: 익명으로 조언 받을 수 있는 시스템
   -타임어택(가칭): 제한 시간 내 빠르게 대화, 가볍게 털어놓기
   -써클: 질문에 대해 순차적으로 발언하는 감정 나눔 모임
   -랜덤 매칭 대화: 익명 대화 파트너와의 즉흥 소통
   -함께 털어놓기: 집단 감정 공유 활동
   -AI 조언 및 위로: 감정 분석 기반으로 공감 메시지 제공
   -AI 감정 정리: 말로 표현하기 어려운 감정 정리를 돕는 기능
 ◦ 보조 기능
   -통계 기능: 이용자 감정 변화 및 사용 패턴 리포트
   -1·2차 필터링 시스템: 욕설/민감정보 차단
   -세대 간 언어 번역기: 공감 소통을 위한 세대 차 해소
   -감성 콘텐츠 추천: 책, 음악, 글귀 공유
   -질문 기반 일기 & 오늘의 질문: 자기 성찰 유도

 ◦ 기술 및 작동 원리
   -AI 기반 조언/감정 분석: LLM을 활용한 실시간 대화 보조
   -RAG 기반 필터링: 기존 사례 및 통계를 기반으로 한 2차 검열
   -가벼운 욕설 필터링 AI: 1차로 표현 제어
   -데이터 분석: 감정 통계, 커뮤니티 피드백 시각화
   -경량화된 시스템 설계: 부담 없이 사용할 수 있는 설계
 ◦ 향후 계획 및 확장 전략
   -기능 고도화: 콘텐츠 자체 품질 향상 및 중독 방지 요소 강화
   -앱 분리 확장: 세대 간 언어 번역기 → 키보드 앱으로 출시
   -시장 테스트 및 피드백 수집: MVP 단계 후 사용자 인터뷰와 반영
   -파트너십 확장: 정신건강 기관, 학교, 청년단체 등과 협업
 ◦ 현실화를 위한 필요 요소
   -LLM 및 감정 분석 AI 기반 시스템 서버
   -DB 구축 및 클라우드 저장소 확보
   -지속적 관리 인력 확보 및 유지 비용 확보
   -사용자 테스트용 앱 MVP 버전 구현

  ◦1단계: 핵심 기능 MVP 개발 (Relpit 활용 기초 구현 + 클로즈 베타)
  ◦ 익명 채팅 시스템 (텍스트 기반)
   -닉네임 자동 생성
   -1:1 또는 익명방 랜덤 매칭
   -Firebase or Replit DB를 이용한 대화 기록 저장
  ◦ 간단한 감정 분석 + 위로 생성
   -ChatGPT API 또는 HuggingFace 모델을 통한 긍·부정 감정 분류
   -감정별 위로 문장 자동 출력 (프롬프트 활용)
  ◦기초 필터링 시스템
   -욕설, 자해 단어 등 키워드 기반 필터링
   -감정 점수 기준 (예: 부정 감정 ≥ 70% → 자동 공감 메시지 출력)
  ◦감정 일기 기능
   -한 줄 감정 상태 기록
   - 시간별 감정 로그 저장
  ◦테스트 계획
   -친구·학교 동료 10~20명 대상 클로즈 베타
   - 구글폼 등으로 사용자 피드백 수집 → 기능 보완 계획 수립
  ◦2단계: 고도화된 감정 케어 기능 개발 (공감형 커뮤니티 설계)
  ◦ AI 공감 대화 기능
   -"오늘 하루 어땠어?" → AI가 질문하고 감정에 따라 위로 / 조언 선택
   -감정 이입 프롬프트 설계 (예: GPT로 “공감형 반응” 유도)
  ◦ 감정 태그 기반 매칭 시스템
   -감정 선택 → 유사 감정을 느낀 사람과 랜덤 대화
   -대화 전 '원하는 대화 방식' 선택 (위로 받고 싶어요 / 그냥 들어줘 등)
  ◦정서 안정 도우미 탭
   -'마음 정리 도구' (짧은 명상 스크립트, 글쓰기 공간, 간단한 감정 해소 질문)
  ◦안전성 강화 : 고급 필터링 시스템 도입
   -위험 대화 탐지 시 자동 응답 + 관리자 알림
   -신고 / 차단 기능 추가

  ◦3단계: 개인화 + 장기 확장 기반 마련
  ◦ 대화에 따른 MBTI 변화 분석
   -AI 대화 스타일 조정: 내성적이면 천천히 질문, 외향적이면 리액션 중심
   ◦ 감정 리포트 기능
   -주간 감정 요약, 가장 많이 느낀 감정, 자주 등장한 키워드
   -‘나를 위로했던 말’, ‘감정 변화 추이’ 등 시각화 제공
   ◦정신건강 관련 기관과 연계
   -앱 내 상담기관 정보 제공
   -위기감지 시 ‘전문가 상담 연결’ 버튼
  ◦수익모델 구상
   -광고 없는 프리미엄 버전
   -공공기관/학교에 라이선스 형태 제공 가능
